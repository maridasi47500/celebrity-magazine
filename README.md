🎥 L'Écosystème : Audiovisuel & Industries Créatives

Ton univers est une métropole médiatique permanente où se croisent paparazzis, stars (riders) et couples influents. C'est un monde où la rumeur et la nouvelle officielle ont le même poids, à condition qu'elles passent par le filtre technologique approprié.
🌎 Le Monopole Linguistique : "English or Translation"

    Source Absolue : Les news et les rumeurs n'existent nativement qu'en anglais.

    Diffusion Live : Ton système sert de moteur de traduction instantanée pour tous les autres supports (TV, Radio, Réseaux Sociaux), rendant l'information mondiale accessible en un clic.

💼 La "LinkedIn-isation" des Réseaux Sociaux

C'est le point le plus original de ton concept : tout est une compétence (skill).

    L'Espace "IN" : Les plateformes fusionnent (Twitter-In, Facebook-In, etc.).

    La Gamification du Pouvoir : Le seul moyen d'agir ou d'interagir sur les réseaux sociaux est de posséder la Skill LinkedIn correspondante. Ton influence est dictée par tes compétences réelles ou virtuelles.

🕴️ Protocole de Vie : Le Code et la Politesse

    Politesse Infaillible : L'usage de l'anglais doit être systématiquement poli, créant une esthétique de communication "haute couture", même dans la vie de tous les jours.

    Publication Codée : On peut parler de tout, mais chaque message doit respecter un code de publication strict ou une forme unique, transformant chaque post en une donnée structurée.
